# HTML Translator GUI

Простое графическое приложение для перевода HTML-файлов с помощью **DeepL API**.  
Позволяет выбрать исходный HTML-файл, указать папку для сохранения перевода и язык перевода.  
Поддерживает сохранение шаблонных переменных (`{var}`, `[[var]]`, `{% var %}`), чтобы они не переводились.

---

## Возможности

- Перевод HTML-файлов в выбранный язык: `EN`, `UK`, `ES`, `RU`, `PT`.
- Сохранение исходной структуры HTML и переносов строк.
- Поддержка шаблонных переменных и тегов, чтобы они не изменялись при переводе.
- Графический интерфейс.
- Выбор папки для сохранения результата.

---

## Требования

- Java JDK 17 или выше
- Maven (для сборки проекта и зависимостей)
- DeepL API ключ

## Сборка

1. Склонируйте репозиторий

```bash 
git clone https://github.com/yourusername/html-translator.git
```

2. Скачайте зависимости
```bash 
mvn clean install
```

---

## Использование

1. Запустите приложение.
2. Введите ваш DeepL API Key.
3. Выберите HTML-файл для перевода.
4. Укажите папку для сохранения перевода.
5. Выберите язык перевода.
6. Нажмите Translate.
7. Дождитесь завершения перевода. Прогресс отображается в прогресс-баре.
8. Готовый файл сохранится в выбранной папке.

---

## Особенности

- Все шаблонные переменные ({var}, [[var]], {% var %}) будут временно заменены на специальные метки, чтобы DeepL их не трогал.
- Приложение переводит только текст внутри HTML, игнорируя теги, скрипты и стили.
- Большие тексты делятся на чанки по 4800 символов, чтобы DeepL API не возвращал ошибки.
